# Identifying nonlinear causal graphs with ICA

## Usage 

1. Clone
```bash
 git clone https://github.com/rpatrik96/nl-causal-representations.git
```

2. Install
```bash
pip3 install -e .
```

3. Run:
```bash
 python3 care_nl_ica/cl_causal.py
```
